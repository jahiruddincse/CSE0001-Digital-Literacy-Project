## 📚 Module Summary

### Task 1 – Infographic
Created a digital literacy infographic using Canva, focusing on safe internet practices, useful tools, and maintaining a professional online presence.

### Task 2 – Digital Portfolio
Built and updated my profiles on platforms like GitHub and LinkedIn to create a basic professional online presence.

### Task 3 – Platforms
Solved a beginner-level coding problem and created a Digital Literacy Awareness Quiz using Google Forms. Responses were linked to Google Sheets for analysis.

### Task 4 – Email Etiquette
Drafted two professional emails (assignment extension and internship inquiry) and created a social media Do’s and Don’ts checklist.

### Task 5 – Cybercrime Awareness
Prepared a case study on UPI payment fraud and created a prevention checklist with practical safety tips for students.
